Readme file to evaluate RAID against network mapping attacks
This setup requires three ECUs
***************************************************************

sendMult1: ECU 1 sending Multiple Periodic IDs
sendMult2: ECU 2 sending Multiple (different) Periodic IDs
scannetworkChangePeriod: Attacker, scans the network and identifies which ECU sends which IDs.


sendMultRAID1: ECU 1 sending Multiple Periodic IDs equipped with RAID
sendMultRAID2: ECU 2 sending Multiple (different) Periodic IDs equipped with RAID
scannetworkChangePeriodRAID: Attacker, scans the network and identifies which ECU sends which IDs.
